# 🚗 Road Accident Analysis Dashboard (Microsoft Excel)

## 📌 Project Overview

This project is an interactive **Road Accident Analysis Dashboard** built entirely in **Microsoft Excel**. The dashboard provides comprehensive insights into road accident casualties, helping users analyze accident trends, severity levels, vehicle involvement, road conditions, and geographical distributions.

The dashboard transforms raw accident data into meaningful visual insights using Excel's powerful data analysis and visualization features.

---

## 🎯 Project Objectives

* Analyze total road accident casualties.
* Monitor fatal, serious, and slight injury trends.
* Compare Current Year (CY) and Previous Year (PY) accident statistics.
* Identify accident-prone road types and conditions.
* Understand vehicle-wise casualty distribution.
* Examine accident patterns by location and lighting conditions.
* Enable interactive filtering for detailed analysis.

## 📈 Key Insights

* Most casualties occurred on **Single Carriageway Roads**.
* Cars accounted for the highest number of casualties.
* Urban areas recorded more casualties than rural areas.
* Dry road conditions showed the highest accident occurrence.
* Slight injuries represented the largest proportion of total casualties.
